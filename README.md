**Commands of Our Bot**

* Now Playing (~np)

* Queue system (~queue, ~q)

* Loop / Repeat (~loop)

* Shuffle (~shuffle)

* Volume control (~volume, ~v)

* Lyrics (~lyrics, ~ly)

* Pause (~pause)

* Resume (~resume, ~r)

* Skip (~skip, ~s)

* Skip to song # in queue (~skipto, ~st)

* Remove song # from queue (~remove, ~rm)

* Toggle pruning of bot messages (~pruning)

* Help (~help, ~h)
